Para actualizar el presupuestador, hay dos cosas independientes que hacer. 
Si se quieren actualizar los precios, de momento es de manera manual desde https://gist.github.com. Ahí hay un archivo tarifas-maderafina.json que se puede modificar o reemplazar totalmente.
Por otro lado, se puede actualizar el presupeustador en sí. Para eso, dentro de github hay que cambiar index.html. El presupuestador que nos da claude lo reemplazamos directamente ahí.
Luego damos a commit changes y el enlace se actualizará. Se puede ver el estado de la subida en Actions.
